# Linux Log Analysis Lab (Planned)

## Status
🚧 In Progress / Planned

## Objective
This lab will focus on analyzing Linux log files and authentication data to identify suspicious activity and support incident response investigations.

## Planned Skills
- Reviewing /var/log/auth.log for authentication events
- Analyzing failed login attempts (btmp)
- Reviewing login history (wtmp / last)
- Identifying suspicious user activity
- Building an incident timeline from logs

## Why This Lab Matters
Linux log analysis is a core skill for SOC analysts and incident responders. Understanding authentication logs, login patterns, and system activity is critical for detecting unauthorized access and investigating security incidents.

## Next Steps
This lab will be completed in a home lab environment using:
- Ubuntu Server
- Kali Linux
- Simulated attack scenarios

Screenshots and full analysis will be added after completion.
