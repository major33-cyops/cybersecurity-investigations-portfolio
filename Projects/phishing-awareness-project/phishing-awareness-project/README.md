# Phishing Awareness Project

## Overview
This project focuses on identifying phishing emails and malicious websites by analyzing common indicators used in social engineering attacks. The goal is to improve threat recognition and promote safe user behavior.

## Objective
- Identify phishing email characteristics
- Recognize malicious website indicators
- Understand social engineering tactics
- Improve cybersecurity awareness

## Tools Used
- Research-based analysis
- Security awareness concepts
- Example phishing scenarios

## Phishing Indicators (Email)

### 1. Suspicious Sender Address
- Email looks legitimate but contains slight misspellings
- Example: support@paypaI.com (capital “i” instead of “l”)

### 2. Urgent or Threatening Language
- “Your account will be locked immediately”
- Creates panic to force quick action

### 3. Unexpected Attachments
- Files that were not requested
- May contain malware

### 4. Mismatched Links
- Link text does not match the actual URL
- Hovering reveals a different destination

### 5. Generic Greetings
- “Dear Customer” instead of your actual name

---

## Phishing Indicators (Websites)

### 1. Fake Login Pages
- Looks identical to real website
- Used to steal credentials

### 2. Incorrect URL
- Slight spelling changes or extra characters
- Example: amaz0n-login.com

### 3. Missing HTTPS or Certificate Issues
- Browser warnings or “Not Secure” label

### 4. Poor Design or Broken Features
- Low-quality images, links that don’t work

1. Researched common phishing attack methods

3. Organized indicators into email vs website categories
4. Created structured awareness content
5. Connected indicators to real-world scenarios

- Created content useful for training and awareness

## Example Scenario
A user receives an email stating their bank account is locked and must be verified immediately.

Steps taken:
1. User checks sender email and notices irregularities  
2. Hovers over link and sees suspicious URL  
3. Does not click the link  
4. Reports email as phishing  
5. Deletes the message
